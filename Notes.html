Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web.

What is HyperText?

HyperText refers to text that contains links to other pages or different parts of the same page. This allows the web to be connected and easy to navigate.

What is a Markup Language?

A Markup Language is a system for annotating text with tags that tell the browser how to display content. In HTML, you “mark up” plain text to give it structure and meaning.

Why Learn HTML?

  • It is the backbone of every website.
  • Understanding HTML is essential before learning CSS or JavaScript.
  • It helps structure your content for browsers and search engines.

Lecture 1: Headings & Paragraphs

1. Headings

Headings are used to define titles and subtitles on your page. Smaller number = more important.

There are six levels of headings:


<h1>Heading 1</h1>  <!-- Most important heading -->
<h2>Heading 2</h2>  <!-- Major section -->
<h3>Heading 3</h3>  <!-- Sub-section under h2 -->
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>  <!-- Least important heading -->

Tips:

  • Use only <h1></h1> for the main title.
  • Use headings to create a hierarchy of content.

2. Paragraphs

The

<p></p> tag is used to group sentences together into a paragraph.


<p>This is the first paragraph.</p>
<p>This is the second paragraph.</p>

Notes: Browsers automatically add spacing between paragraphs, so avoid using multiple line breaks manually.

3. Combining Headings and Paragraphs

You can combine headings and paragraphs to structure your notes:


<h1>My HTML Notes</h1>
<p>HTML is the standard language for creating web pages.</p>

<h2>Why Learn HTML?</h2>
<p>It helps structure content and is the backbone of websites.</p>
