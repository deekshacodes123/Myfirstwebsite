HTML : Hyper Text Markup Language

HTML stands for Hyper Text Markup Language. It is the standard language used to create and structure the content of a webpage.

  • HyperText: Create links that connect web pages, making the web a connected "web" of information.
  • Markup Language: Use "tags" to give content structure. You are "marking up" a plain text document.
