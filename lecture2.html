Lecture 2: Line Breaks, Horizontal Rule & Lists

Welcome to Lecture 2! In this lecture, we will learn about three important HTML elements: line breaks, horizontal rules, and lists. These elements help make your webpage organized and easy to read.

1. Line Breaks (br)

The br tag is used to move text to the next line without starting a new paragraph. This is useful for addresses, poems, or any text that requires single line breaks.

Example:

My address is:
221B Baker Street
London, England

2. Horizontal Rule (hr)

The hr tag inserts a horizontal line across the page. It is used to separate different sections, making your content look clean and organized.

Example:

This is the first topic.

This is the next topic, separated by a line.

3. Lists

Lists are used to group related items together. HTML provides two types of lists: unordered lists and ordered lists.

Unordered List (ul)

An unordered list uses bullets to display items.

  • Apples
  • Bananas
  • Oranges

Ordered List (ol)

An ordered list uses numbers to display items.

  1. Wake up early
  2. Brush your teeth
  3. Have breakfast

5. Images Example

Images make your webpage visually appealing. Use the img tag with src and alt attributes.

This concludes Lecture 2. Practice these tags by creating your own examples. Remember, br is for line breaks, hr is for horizontal lines, ul and ol are for lists, and img is for images.
