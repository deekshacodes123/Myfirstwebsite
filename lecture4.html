Forms in HTML

The Web Needs Interaction

A website is not just a page to read. It needs a way to get information from the user and send it to the server.

Without this, you cannot log in, search, post comments, or buy anything online.

How Forms Solve This

Forms let us:

  1. Show fields for the user to fill (text boxes, checkboxes, dropdowns).
  2. Package the user’s data neatly.
  3. Send the data to the server.
  4. Specify how the data should be sent.
